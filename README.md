# Detecao_de_Anomalias_transacc
📊 Detecção de Fraude em Cartões de Crédito
Este projeto tem como objetivo analisar e construir um modelo de aprendizado de máquina (Machine Learning) para identificar transações fraudulentas em cartões de crédito.

O projeto está sendo desenvolvido no Google Colab utilizando a linguagem Python e as principais bibliotecas de Ciência de Dados.

🛠️ Tecnologias e Ferramentas
Python 3

Pandas: Para manipulação e análise de dados.

Google Colab: Ambiente de desenvolvimento em nuvem.

📁 Sobre os Dados
O conjunto de dados utilizado foi obtido diretamente do repositório do TensorFlow (creditcard.csv). Ele contém transações financeiras realizadas por titulares de cartões europeus.

Características (V1 a V28): Por motivos de confidencialidade, os dados originais passaram por uma transformação matemática chamada PCA (Análise de Componentes Principais), transformando as colunas originais nesses recursos numéricos.

Time: O tempo em segundos decorrido entre cada transação e a primeira transação do banco de dados.

Amount: O valor da transação.

Class: A nossa variável alvo (target). Onde 0 representa uma transação normal e 1 representa uma fraude.
